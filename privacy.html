Privacy Policy — NanoClaw (Personal Use)

Last updated: 2026-05-05

This app is operated for personal use only by a single user (the developer) and does not collect, store, share, or sell data from any other person.

Data accessed: the app reads only the developer's own TikTok account metadata (video list, view/like/comment/share counts) via the TikTok Display API, using the developer's own access token. No data from other TikTok users is accessed, requested, or stored.

Data storage: retrieved data is stored locally on the developer's private machine in a SQLite database for the developer's own analytics. The data is never transmitted to a third party, published, or used for advertising.

Third parties: none.

Cookies / tracking: none.

Children: the app is not directed at children. The developer is over 18.
